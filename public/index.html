This backend express/mysql app includes the following routes:

GET /users
will provide a list of all the users from the database with an id, first and last name, address, city, county, state, zip, primary phone, secondary phone, and email

GET /users/:id
will provide you with a single user by id, provided they exist

POST /users
will allow you to add an entry for a new user that will be assigned an ID and returned to you as long as you provide the first and last name, address, city, county, state, zip, primary phone, secondary phone, and email

PUT /users/:id
will allow you to update a specific user's first and last names with the new information specified in the body of the request, provided they exist

DELETE /users/first_name
will allow you to delete a user or users from the database by specifying the first name in the body of the request

Test the routes in Postman
